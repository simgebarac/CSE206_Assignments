### 2. [Assignment 2: Word Chain Game](./CSE206_Assignment%232)
* **Technology**: React.js .
* **Description**: An interactive game project where the user must derive a new word starting with the last letter of the previous word.
* **Features**:
    * **Validation**: Dictionary check for valid words, prevention of duplicate words, and character matching algorithms.
    * **Feedback**: Dynamic color-coded warning messages (Green for correct, Orange for used, Red for invalid).
    * **Mechanics**: 15-second move limit, scoring system, and high score tracking using **LocalStorage**.

### 3. [Assignment 3: Cinema Web Management System](./CSE206_Assignment%233)
* **Technology**: ASP.NET Core MVC (C#) & SQL Server.
* **Description**: A comprehensive web application covering cinema management processes (Movies, Theaters, Tickets) connected to a database.
* **Features**:
    * **CRUD Operations**: Full management (Create, Read, Update, Delete) for Movies, Theaters, and Ticket data.
    * **User Management**: Identity infrastructure for user registration and login systems.
    * **Database**: Optimized data retrieval using SQL **Stored Procedures** (e.g., for Popular Movies).
    * **Communication**: Passing dynamic success/error messages from Controller to View via **TempData**.
